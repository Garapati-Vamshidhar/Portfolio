MONITORING WITH NAGIOS(April '03 - April '08, 2023)

Context:

The "Monitoring with Nagios" project serves a crucial need in maintaining the stability and optimal performance of network services, hosts, and applications. By implementing Nagios as a monitoring solution, the project addresses the critical requirement of ensuring system availability. Nagios actively monitors the health and responsiveness of various components, providing real-time visibility into their status. Nagios facilitates the tracking and analysis of key performance metrics such as CPU usage, memory utilization, network traffic, and disk space. This data empowers administrators to identify performance bottlenecks, plan for capacity expansion, and optimize resource allocation.

Movie Listing application Deployment in cloud infrastructure(Mar '12 - Mar '27, 2023)

Context:

The Movie Listing Application Deployment in Cloud Infrastructure project is undertaken to address the challenges associated with managing and presenting movie information in a scalable and reliable manner. With the ever-growing movie industry and increasing user demands for up-to-date information, traditional methods of accessing movie listings have become outdated and inefficient. To overcome these challenges, the project aims to leverage the benefits of cloud infrastructure. By deploying the movie listing application on the cloud, it becomes accessible to a larger audience without geographical limitations. Cloud infrastructure offers scalability, flexibility, and reliability, allowing the application to handle increasing user traffic and data loads effectively.

Tools Used:

1. MongoDB
2. AWS S3
3. Git/GitHub
4. Docker
5. Elastic Loadbalancer
6. Amazon EC2

Skills:

 Strong team player with experience working collaboratively with team members, developers, and stakeholders to understand pain points, gather requirements, and implement process improvements effectively.
 Skilled in managing end-to-end process improvement projects, including defining project scope, creating project plans, coordinating resources, and tracking progress to ensure timely and successful implementation of proposed solutions.
 Demonstrated ability to analyze the movie listing application deployment process, identify bottlenecks, and inefficiencies that impact scalability, reliability, or user experience.
